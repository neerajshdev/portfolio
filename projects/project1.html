Architecture

This application was built using the MVVM (Model-View-ViewModel) architecture pattern, which provides a clean separation of concerns and improves testability.

Technologies Used

  • Kotlin - Primary programming language
  • Android Jetpack - Suite of libraries and tools
  • Room Database - Local data persistence
  • Retrofit - Network API calls
  • Coroutines - Asynchronous programming
  • LiveData & ViewModel - Lifecycle-aware data handling
  • Navigation Component - In-app navigation
  • Material Design Components - UI elements
